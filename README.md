# compiled_languages_faceoff
A faceoff between compiled languages

## Tests

1. Vector reductions: `s = s + a * v_i`
1. Vector arithmetic: `x_i = x_i  + a * y_i`
1. Vector dot-products: `s = s + x_i * y_i`
1. Intrinsic square-root: `s = s + sqrt(v_i)`
1. Intrinsic power: `s = s + v_i ** 2`
1. Intrinsic natural-exponent: `s = s + exp(v_i)`
1. Intrinsic logarithm: `s = s + ln(v_i)`
1. Intrinsic trigonometry: `s = s + sin(v_i)`
